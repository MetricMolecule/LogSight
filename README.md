# LogSight

A lightweight distributed log ingestion and analytics platform built with FastAPI, Redis Streams, PostgreSQL and Docker.

## Features

- Batched log ingestion
- Redis Streams pipeline
- Background workers
- Full-text log search
- Live log tail
- Alert engine

## Tech Stack

- Python
- FastAPI
- Redis Streams
- PostgreSQL
- Docker
- Server-Sent Events
